# Code of Conduct

## Our Commitment

This project is committed to a respectful, professional, and constructive community.
Contributors, maintainers, and users are expected to collaborate in good faith and
treat each other professionally.

## Expected Behavior

- Be respectful in language and tone.
- Assume positive intent, ask clarifying questions, and focus on facts.
- Give and receive feedback constructively.
- Keep technical discussions on-topic and actionable.
- Work constructively with contributors of all experience levels.

## Unacceptable Behavior

- Harassment, abuse, threats, or discriminatory language.
- Personal attacks, trolling, or deliberately inflammatory behavior.
- Repeated off-topic disruption of issues, pull requests, or discussions.
- Sharing private information without explicit permission.
- Any behavior that creates hostility or disrupts collaboration.

## Scope

This Code of Conduct applies across all project spaces, including:

- GitHub issues and pull requests
- Discussions and review comments
- Any public or private communication tied to this repository

## Reporting

If you experience or witness behavior that violates this Code of Conduct, report it
to the maintainers by opening a private security advisory or contacting the repository
owners directly.

Please include:

- What happened
- Where it happened (link if possible)
- When it happened
- Any relevant context or evidence

All reports will be reviewed promptly and handled as confidentially as possible.

## Enforcement

Maintainers may take any action they deem appropriate, including warning, content
removal, temporary suspension, or permanent ban from participation.
